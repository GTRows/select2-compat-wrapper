Select2 Compat Wrapper 4.0.13 + Wrapper

All components below use 3.3.x-style API calls running on Select2 4.0.13 via the compatibility wrapper.

Basic Select

Standard initialization with placeholder and allowClear. Uses 3.3.x API.
$el.select2({ placeholder: "Select a country...", allowClear: true });

Data Setter

Programmatically set value using select2("data", obj) - a method removed in 4.x.
$el.select2("data", { id: "jp", text: "Japan" });

Container Access

Access the internal container via select2("container") - removed in 4.x.
var $c = $el.select2("container"); $c.css("border", "2px solid #f0883e");

Multiple Select with maximumSelectionSize

Uses the 3.3.x option name "maximumSelectionSize" (renamed to maximumSelectionLength in 4.x).
$el.select2({ maximumSelectionSize: 3, placeholder: "Pick up to 3..." });

Custom Formatting (formatResult / formatSelection)

Uses 3.3.x formatResult and formatSelection options (renamed to templateResult/templateSelection in 4.x).
$el.select2({ formatResult: function(item) { return "<b>" + item.text + "</b>"; }, formatSelection: function(item) { return "[" + item.text + "]"; } });

Custom Matcher (old signature)

Uses 3.3.x matcher(term, text, option) signature. Try typing "an" to match "Germany" and "Japan".
$el.select2({ matcher: function(term, text, opt) { return text.toLowerCase().indexOf(term.toLowerCase()) >= 0; } });

Language Functions (formatNoMatches, formatSearching)

Uses 3.3.x format functions (moved to language object in 4.x). Type something that does not match to see.
$el.select2({ formatNoMatches: function() { return "No results found."; }, formatSearching: function() { return "Searching..."; } });

AJAX (3.3.x config shape)

Uses quietMillis, data(term, page), and results(data, page) - all renamed/restructured in 4.x. Searches GitHub repositories.
$el.select2({ ajax: { url: "https://api.github.com/search/repositories", quietMillis: 400, data: function(term, page) { return { q: term }; }, results: function(data, page) { return { results: data.items.map(...), more: false }; } } });

Event Bridge

Listens for 3.3.x-style events (select2-open, select2-close, select2-selecting, etc.) which are bridged from 4.x events.
$el.on("select2-open", function() { ... }); $el.on("select2-close", function() { ... }); $el.on("select2-selecting", function(e) { log(e.choice); });
